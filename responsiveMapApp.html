Responsive mapping application

Information architecture - Wireframing - Visual Design - Styles Implementation

The ideas of adaptive web design have been incorporated in this project implemented at the Esri Application Prototype Lab to create a mapping application which responds to the resolution of the display window. It re-configures the UI elements to enable the same application to function in a similar fashion on most devices. The application can be embedded in a HTML iframe tag and would adjust according to the specified size.   Demo and download link

Project in collaboration with Sathya Prasad and Kiril Vatev at Esri Applications Prototype Lab
